## 📌 Project Overview

This is a personal Power BI project that visualizes **Sales & Revenue data** using interactive dashboards and reports. The goal of this project is to analyze sales trends, track revenue performance, and derive meaningful business insights through data visualization.

---

## 🎯 Objectives

- Analyze overall sales and revenue performance
- Identify top-performing products/categories
- Track monthly and yearly sales trends
- Visualize key metrics using interactive charts and graphs

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|------|---------|
| **Power BI Desktop** | Data visualization & dashboard creation |
| **Microsoft Excel / CSV** | Data source |
| **DAX** | Calculated measures and KPIs |
| **Power Query** | Data cleaning and transformation |

## 📈 Key Insights & Features

- ✅ **Total Sales & Revenue KPIs** — at-a-glance summary cards
- ✅ **Monthly Sales Trend** — line chart showing growth over time
- ✅ **Top Products/Categories** — bar chart ranking best performers
- ✅ **Regional Sales Breakdown** — map or bar chart by location
- ✅ **Interactive Filters** — slicers for date, category, and region
